hexo博客中插入的图片不显示问题的解决办法

hexo博客如果采用本地间接引用的方法插入图片的话，需要事先进行一定的配置，不然就会出现不显示的问题。

安装插件

在hexo文件路径下，用如下命令安装即可

npm install hexo-filter-image --save

修改文件

打开_config.yml文件，将post_asset_folder语句由false改成true

修改后，当你创建新文章title.md时，会在同一路径下自动生成一个名为title的文件夹，我们需要将文章中涉及到的图片放入该文件夹内

图片路径的引用

注意引用格式，如想引用图片1.png，正确格式为![name](./title/1.png)

其他尝试

如果以上方法试过仍不能解决，可能是插件中的代码存在一点问题，需要做适当调整。
